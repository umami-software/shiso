## Config file

Shiso uses two configuration files:

- `shiso.config.json` for app-level settings
- `docs.json` (Mintlify format) for docs navigation

## App configuration (`shiso.config.json`)

```json
{
  "contentDir": "/path/to/content",
  "docsConfigPath": "./src/docs.json",
  "docs": {
    "title": "Docs - Shiso"
  },
  "blog": {}
}
```

## Docs configuration (`docs.json`)

`docs.json` uses Mintlify's navigation format:

```json
{
  "$schema": "https://mintlify.com/docs.json",
  "name": "Shiso",
  "navigation": {
    "tabs": [
      {
        "tab": "Documentation",
        "groups": [
          {
            "group": "Getting started",
            "pages": ["index", "installation", "configuration"]
          },
          {
            "group": "Writing content",
            "pages": ["writing-content", "text", "images", "lists", "code-blocks"]
          }
        ]
      },
      {
        "tab": "Components",
        "groups": [
          {
            "group": "Components",
            "pages": ["components/index"]
          }
        ]
      }
    ]
  }
}
```

Each page entry maps to a file under `contentDir/docs`:

- `"index"` -> `src/content/docs/index.mdx`
- `"installation"` -> `src/content/docs/installation.mdx`
- `"components/index"` -> `src/content/docs/components/index.mdx`
