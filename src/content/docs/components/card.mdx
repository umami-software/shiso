title: Card
---

Use `Card` and `CardGroup` to present grouped links or summaries.

## Supported props

### `Card`

- `title?: ReactNode`
- `href?: string`
- `icon?: ReactNode`
- `arrow?: boolean` (defaults to `true` when `href` exists)
- `children?: ReactNode`

### `CardGroup`

- `cols?: 1 | 2 | 3 | 4` (defaults to `2`)
- `children?: ReactNode`

## Basic usage

```tsx
<Card title="API Reference" href="/docs/components/response-field">
  Learn how to model response schemas.
</Card>
```

<Card title="API Reference" href="/docs/components/response-field">
  Learn how to model response schemas.
</Card>

## Card group

<CardGroup cols={2}>
  <Card title="Getting Started" href="/docs/installation">
    Install and configure Shiso.
  </Card>
  <Card title="Components" href="/docs/components">
    Browse all docs components.
  </Card>
</CardGroup>

## Behavior notes

- cards with `href` render as links
- cards without `href` render as static containers
