title: Callout

Use callouts to highlight important content blocks.

## Generic callout

```tsx
<Callout variant="note" title="Heads up">
  This is a custom callout body.
</Callout>
```

<Callout variant="note" title="Heads up">
  This is a custom callout body.
</Callout>

## Variants

### Note

<Note>Use this for neutral supplemental information.</Note>

### Tip

<Tip>Use this for best practices and helpful shortcuts.</Tip>

### Warning

<Warning>Use this for risky operations or destructive behavior.</Warning>

### Info

<Info>Use this for contextual details users should know.</Info>

### Check

<Check>Use this for success or completion states.</Check>
