title: Frame
---

Use `Frame` to visually contain custom content with an optional caption.

## Basic usage

```tsx
<Frame caption="Environment variables in `.env`">
  <code>NEXT_PUBLIC_API_URL=https://api.example.com</code>
</Frame>
```
