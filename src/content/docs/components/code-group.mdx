title: Code Group
---

Use `CodeGroup` to render multiple code snippets behind tabs.

## Basic usage

````tsx
<CodeGroup>
  ```bash npm
  npm install @umami/shiso
  ```

  ```bash pnpm
  pnpm add @umami/shiso
  ```
</CodeGroup>
````
