Use `Tabs` and `Tab` to show alternate content in the same space.

## Supported props

### `Tabs`

- `group?: string`
- `children?: ReactNode`

### `Tab`

- `title?: ReactNode`
- `children?: ReactNode`

## Basic usage

```tsx
<Tabs>
  <Tab title="JavaScript">
    <p>Install with npm.</p>
  </Tab>
  <Tab title="Python">
    <p>Install with pip.</p>
  </Tab>
</Tabs>
```

## Persist selection with `group`

Tabs with the same `group` share selection via local storage.

```tsx
<Tabs group="language">
  <Tab title="Node">...</Tab>
  <Tab title="Go">...</Tab>
</Tabs>
```

## Behavior notes

- tab selection is stored under `localStorage` key `shiso-tabs:<group>`
- one tab child renders as plain content (no tab UI)
