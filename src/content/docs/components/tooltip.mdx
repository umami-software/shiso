---
title: Tooltip
---

Use `Tooltip` to provide short inline hints.

## Supported props

- `tip?: string`
- `children?: ReactNode`

## Basic usage

```tsx
<Tooltip tip="Used to identify the current API key">API key</Tooltip>
```

The value of <Tooltip tip="Used to identify the current API key">API key</Tooltip> must be kept secret.

## Behavior notes

- tooltip text is rendered via native `title` attribute
