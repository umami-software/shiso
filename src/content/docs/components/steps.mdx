title: Steps

Use `Steps` and `Step` for ordered walkthroughs.

## Supported props

### `Steps`

- `children?: ReactNode`

### `Step`

- `title?: ReactNode`
- `children?: ReactNode`

## Basic usage

```tsx
<Steps>
  <Step title="Install dependencies">Run your package manager install command.</Step>
  <Step title="Configure docs">Set navigation in `shiso.config.json`.</Step>
  <Step title="Start the app">Run the local development server.</Step>
</Steps>
```

## Behavior notes

- if no `title` is provided, fallback titles are `Step 1`, `Step 2`, ...
