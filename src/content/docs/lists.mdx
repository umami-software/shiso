## Ordered Lists

To create an ordered list, add line items with numbers and periods.

```markdown
1. First item
2. Second item
3. Third item
```

1. First item
2. Second item
3. Third item

## Unordered Lists

To create an unordered list, add line items starting with the character `-`, `*` or `+`.

```markdown
- First item
- Second item
- Third item
```

- First item
- Second item
- Third item
