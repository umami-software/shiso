## Inline Code

Use backticks for inline code.

```text
Use `inline code` in normal text.
```

## Fenced code blocks

Use triple backticks and include a language for syntax highlighting:

```javascript
function hello(name) {
  console.log('Hello', name);
}
```

````text
```javascript
function hello(name) {
  console.log('Hello', name);
}
```
````

## Highlighting behavior

This app uses:

- `rehype-highlight` for tokenized code output
- a Highlight.js stylesheet imported in `src/app/layout.tsx`

Code colors come from the imported Highlight.js theme.

## Code groups

Use `CodeGroup` when you want tabbed snippets:

````mdx
<CodeGroup>
  ```bash npm
  npm install @umami/shiso
  ```

  ```bash pnpm
  pnpm add @umami/shiso
  ```
</CodeGroup>
````
