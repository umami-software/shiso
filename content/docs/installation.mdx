---
title: Installation
---

This guide sets up Shiso.

## Prerequisites

- Node.js 20+
- pnpm (or npm/yarn)

## 1. Install dependencies

```bash
pnpm install
```

## 2. Confirm key files

```text
.
├── content
│   └── docs
│       └── index.mdx
└── src
    ├── app
    │   └── docs
    │       └── [[...slug]]
    │           └── page.tsx
    └── docs.json
```

## 3. Configure navigation

Edit `src/docs.json`:

```json
{
  "$schema": "https://mintlify.com/docs.json",
  "theme": "mint",
  "name": "My Docs",
  "navigation": {
    "tabs": [
      {
        "tab": "Documentation",
        "groups": [
          {
            "group": "Getting started",
            "pages": ["index", "installation"]
          }
        ]
      }
    ]
  }
}
```

## 4. Create matching content files

Each page slug in `navigation` must map to a file in `content/docs`:

```text
content/docs/index.mdx
content/docs/installation.mdx
```

## 5. Run locally

```bash
pnpm dev
```

Open `/docs` and confirm:

1. content renders
2. side navigation matches `src/docs.json`
3. prev/next links follow navigation order
4. heading anchors appear in "On this page"
