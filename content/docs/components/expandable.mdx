---
title: Expandable
---

Use `Expandable` for collapsible sections.

## Supported props

- `title: ReactNode`
- `defaultOpen?: boolean`
- `children?: ReactNode`

## Basic usage

```tsx
<Expandable title="Advanced settings">
  These settings are optional and can be left at defaults.
</Expandable>
```

<Expandable title="Advanced settings">
  These settings are optional and can be left at defaults.
</Expandable>

## Open by default

```tsx
<Expandable title="Open section" defaultOpen>
  This section starts expanded.
</Expandable>

## Behavior notes

- each `Expandable` is rendered as a single-item accordion
```

<Expandable title="Open section" defaultOpen>
  This section starts expanded.
