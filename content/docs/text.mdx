## Headings

Use `#` levels to structure your page and generate anchor navigation.

```text
# Title

## Section

### Subsection
```

## Emphasis

Use standard Markdown emphasis:

```markdown
Use **bold** for strong emphasis.
```

```markdown
Use _italic_ for light emphasis.
```

```markdown
Use ~~strikethrough~~ when something is outdated.
```

## Tables

GitHub-flavored markdown tables are supported:

```markdown
| Name | Type |
| ---- | ---- |
| id   | string |
| age  | number |
```
