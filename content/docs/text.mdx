## Headers

You can define headers by using the `#` symbol.

```text
# Title

## Section

### Subsection
```

## Text Formatting

To make text **bold**, wrap the text with `**` characters.

```markdown
To make text **bold**, wrap the text with `**` characters.
```

To make text _italic_, wrap the text with `_` characters.

```markdown
To make text _italic_, wrap the text with `_` characters.
```

To make text ~strikethrough~, wrap the text with `~` characters.

```markdown
To make text ~strikethrough~, wrap the text with `~` characters.
```
