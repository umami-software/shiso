## Ordered Lists

Use numeric markers:

```markdown
1. First item
2. Second item
3. Third item
```

1. First item
2. Second item
3. Third item

## Unordered Lists

Use `-`, `*`, or `+` markers:

```markdown
- First item
- Second item
- Third item
```

- First item
- Second item
- Third item

## Nested Lists

```markdown
1. Parent
   - Child A
   - Child B
2. Next parent
```

1. Parent
   - Child A
   - Child B
2. Next parent

## Task Lists

GitHub-flavored markdown task lists are supported:

```markdown
- [x] Build docs route
- [x] Add navigation to `src/docs.json`
- [ ] Write more examples
```

- [x] Build docs route
- [x] Add navigation to `src/docs.json`
- [ ] Write more examples
